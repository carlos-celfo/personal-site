My Portfolio

Sticky Notes App

React

App that allows to write in post-it types of notes. It has the capacity to add new notes, delete notes that already exist and search for specific notes throught the search bar, doesn't matter if its in the title of the note or the description part. This project- which is part of the Skillcrush online course curricula- represented the first big project I did with React and in which I gained lots of experience about functional components and lifecyle methods, among other React tools.

Guess the word Game

JavaScript, HTML5 and CSS

Game app in which you need to find out which word is hiding behind placeholders (commonly known as "the hanged man"). With this app, I used an API with more than 100 words to show random words everytime the game is started, making the game more interesting. It recognizes also if the input typed by the user is not a string (for example a number or a symbol), or if the input value is already repeated. Besides, there's the possibility to play again, whether you loose or win.
Go and try it! Have fun!

Github repo gallery

Javascript, HTML5 and CSS

App to show my repositories of github in a more simplistic and direct manner. It has the capacity to search an specific repository by their name, so it can be easy to find. an API in it brings the information from github to the app, both of my profile in the platform and also of all of the repositories I've made. Once you choose one project, one click in a button can redirect you the repo in the github page.
If you wanna know my work, I invite you to have a look at it!

Unplugged retreat project

HTML5 and CSS

This is a special website, since it was my first one that is fully responsive and with different pages within, built with flexbox. this website is part of the Skillcrush curricula in order to learn responsive design. As such, is has the capacity to be seen in every screentype and be adapted to it- changing its size depending on the media query declared in the CSS. With this project, I learn to use the web developer tools and realize how useful it is in order to create awesome websites.
